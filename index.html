Project 2
ATHLETICARE

The project is a web-based platform for athletes to connect with doctors. The system allows athletes to enter their injury information to view a list of expert doctors in that area, along with their availability and ratings. Athletes can schedule appointments with doctors based on their ratings and availability and rate the doctors after their sessions. The goal is to improve upon the current state-of-the-art appointment scheduling system by creating a more user-friendly website. In the future, the plan is also to develop a cross-platform app that will be compatible with Android and iOS devices.

Project 5
Badger Chat:

I developed a robust login and registration system using React and React Native, ensuring secure user authentication, error handling, and real-time feedback. Dynamic landing page and chatroom functionality were implemented, enabling seamless navigation between chatrooms, message viewing, and interactive features such as post creation and message refreshing. Integration of the third-party library, react-native-modal, enhanced the user experience for creating posts within specific chatrooms. Secure storage and deletion of JWT tokens were implemented using a secure store for reliable user authentication. Additionally, guest access for anonymous users with a streamlined conversion process was incorporated to encourage registration.
